lu-latex
========

Mans izmantotais LaTeX šablons bakalaura darbam.

## Linux

Nepieciešams:

* Nezinu precīzi, uzliku `texlive-full`

Būvējams ar `make`


## Windows

Nepieciešams:

* `basic-miktex.exe` (bing it ;) )  + instalācija ieķeksēt, lai novelk trūkstošās pakotnes (ja pareizi atceros)

Būvējams ar `build.bat`
